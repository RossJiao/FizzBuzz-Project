# FizzBuzz-Project

## Overview
A classic programming exercise that prints numbers from 1 to 100 with a twist: multiples of 3 become "Fizz", multiples of 5 become "Buzz", and multiples of both become "FizzBuzz".

## Background
FizzBuzz is a common interview question used to test basic programming logic. It demonstrates understanding of:
- Loops (`for` loop)
- Conditional statements (`if`/`elif`/`else`)
- Modulo operator (`%`) for checking divisibility
- Print formatting

## Problem Statement
Create a program that prints each number from 1 to 100 on a new line, with the following substitutions:

| Condition | Output |
|-----------|--------|
| Multiple of 3 | `Fizz` |
| Multiple of 5 | `Buzz` |
| Multiple of 3 AND 5 | `FizzBuzz` |
| Not a multiple of 3 or 5 | The number itself |
